🚀 ML GitOps Automation Platform

This project demonstrates a production-style DevOps GitOps pipeline for deploying a FastAPI machine learning application using Kubernetes. It automates Docker builds, CI/CD workflows, GitOps deployments using ArgoCD, and provides real-time monitoring through Prometheus and Grafana.

Tech Stack: Docker • Kubernetes • ArgoCD • GitHub Actions • FastAPI • Prometheus • Grafana • GitOps

🚀 Open ML App ❤️ Health Check 📊 Metrics 📂 GitHub Repository 📘 Project Documentation

⚙ DevOps Features

  • Automated CI pipeline using GitHub Actions
  • Dockerized ML API with container-based deployment
  • GitOps-based Kubernetes deployment using ArgoCD
  • Rolling updates and self-healing Kubernetes pods
  • Real-time observability with Prometheus & Grafana
  • Production simulation including load testing and scaling
